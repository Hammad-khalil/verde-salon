# **App Name**: VerdePress Salon

## Core Features:

- Minimal Salon Website Pages: Display the Home, Services, and Blog pages with all specified sections including an animated hero, services preview, featured work, testimonials, CTAs, Instagram preview, and comprehensive blog/services listings and detail pages.
- Intuitive CMS Admin Interface: Provide a secure, user-friendly dashboard for non-technical team members to manage all website content and settings.
- Dynamic Content Editing: Allow editors to easily modify and update text, images, videos, buttons, and individual sections directly within the CMS.
- Flexible Drag-and-Drop Page Builder: Enable editors to add, remove, and reorder page sections and modules (e.g., FAQs, galleries, forms) using a visual drag-and-drop interface.
- AI-Enhanced Content Generation Tool: Utilize generative AI as a tool to assist with creating SEO metadata (titles, descriptions, image ALT tags) and suggesting blog post content or sections based on provided keywords or existing page content.
- Global Branding Controls: Centralized settings within the CMS for updating the logo, brand colors, typography, hero images, and background images site-wide.
- Integrated Performance & Analytics Configuration: Offer one-click tools for image optimization (compression, WebP conversion), CSS/JS minification, lazy loading settings, cache control, and input fields for Google Analytics, Google Tag Manager, and Facebook Pixel IDs.

## Style Guidelines:

- The palette is inspired by nature and sophistication. Primary color: A muted forest green (#558C61) for an elegant, serene brand identity. Background color: A very light, desaturated green-gray (#EFF2F0) to ensure a clean, expansive feel. Accent color: A brighter, herbaceous green (#A9DD68) to draw attention to interactive elements and highlights.
- Headline font: 'Playfair' (serif) for a luxurious, high-end feel with strong visual appeal. Body font: 'PT Sans' (sans-serif) for clean readability and a modern, approachable touch in longer text passages.
- Utilize a consistent set of minimal, clean line icons that complement the high-end and minimalist aesthetic, ensuring clear communication without clutter.
- Emphasize spacious, clean layouts with generous whitespace to promote a sense of calm and luxury. Implement a clear, intuitive grid system for showcasing services, work portfolios, and blog posts.
- Incorporate subtle, smooth animations for the hero section as requested, and apply elegant fade-ins or gentle transitions for element visibility and state changes to enhance user experience without being distracting.
